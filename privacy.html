Политика конфиденциальности

Мы уважаем вашу конфиденциальность и обязуемся защищать ваши персональные данные.

1. Сбор данных
Мы собираем только имя и номер телефона для связи по вопросам трудоустройства.

2. Использование данных
Ваши данные используются исключительно для обратной связи и подбора вакансий.

3. Передача данных
Мы не передаём ваши данные третьим лицам без вашего согласия.

4. Хранение данных
Данные хранятся в зашифрованном виде и удаляются по вашему запросу.

5. Контакты
По вопросам удаления данных пишите: privacy@oboronprom.ru
